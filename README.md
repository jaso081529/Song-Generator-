# JJ 674 – Ultra Song Lab (Offline, iPad‑ready)

Single‑file Web‑App mit Live‑Playback, MP3/Mic, FX‑Rack (Reverb/Delay/Compressor), Presets, Humanize, Tap‑Tempo,
Lyrics (TXT/Print), WAV‑Export (Mix) und Stems‑Export – komplett offline, ideal für iPad Safari.

## GitHub Pages
1) Repo anlegen → Dateien hochladen (mind. `index.html`).
2) Settings → Pages → Deploy from a branch → Branch `main` → Folder `/`.
3) Warten, bis URL aktiv ist.
